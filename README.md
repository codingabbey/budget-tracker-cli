# Budget Tracker CLI (Python)

A simple command-line budget tracker to record income/expenses, categorise spending, and view summaries.

## Features
- Add income and expense transactions
- Store data locally in JSON
- View summaries (all-time or by month)
- Category breakdown for expenses
- List and delete transactions

## Requirements
- Python 3.10+ recommended

## Run
```bash
python3 budget_tracker.py
